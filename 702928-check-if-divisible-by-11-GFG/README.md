# [Check if divisible by 11](https://www.geeksforgeeks.org/problems/check-if-divisible-by-114724/1)
## Easy
Given a number s . Check whether it is divisble by 11 or not.
Examples:
Input: s = 76945
Output: true
Explanation: The number is divisible by 11 as 76945 % 11 = 0.

Input: s = 12
Output: false
Explanation: The number is not divisible by 11 as 12 % 11 = 1.

Constraints:1 ≤ |s| ≤ 101000+5
