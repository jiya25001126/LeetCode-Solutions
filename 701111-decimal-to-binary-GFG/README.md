# [Decimal to binary](https://www.geeksforgeeks.org/problems/decimal-to-binary-1587115620/1)
## Easy
Given a decimal integer n, convert it and return its binary equivalent as a string.
Examples :
Input: n = 12
Output: 1100Explanation: The binary representation of 12 is "1100", since 12 = 1×23 + 1×22 + 0×21 + 0×20
Input: n = 33
Output: 100001Explanation: The binary representation of 33 is "100001", since 33 = 1×25 + 0×24 + 0×23 + 0×22 + 0×21 + 1×20

Constraints:1&nbsp;≤ n ≤ 231 - 1
